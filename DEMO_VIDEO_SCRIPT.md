## 技术要点详细说明

### 算法服务细节
1. **ImageRetrieval (I2I)**:
   - 模型路径: `analyze_valid/base_distill_patchmask/augment_base_distill_float32_NWEP_RESISC45_256_2024-09-21-13-44-39/base_distill_NWEP_RESISC45_256_0.9648.pt`
   - 支持模型: base_cnn, base_vit, base_vit_plus, base_distill, base_distill2
   - 性能指标: NWEP_RESISC45 0.965, AID 0.9727, UCMD 0.9856

2. **MedicalRetrieval (T2I)**:
   - 模型路径: `Ksample4Ratio0.32025-01-29/21-09-16/checkpoints/model-best.tar`
   - 医学优化: CLIP医学变体、临床BERT tokenizer
   - 数据集: MIMIC-CXR、VinDr-CXR、RSNA Pneumonia

3. **VideoRetrieval (T2V)**:
   - 模型路径: `ckpts/model_step_10547.pt`
   - 视频处理: 长视频切片、关键帧提取、时空注意力
   - 特征维度: 视频特征和文本特征对齐

### 后端配置要点
- 配置文件: `backend/app/core/config.py`
- 关键配置:
  ```python
  algorithm_mode = "http"  # 或 "deterministic"
  algorithm_gateway_url = "http://127.0.0.1:18080"
  sqlite_path = "data/sqlite/app.db"
  faiss_dir = "data/faiss"
  local_job_workers = 4
  default_topk = 10
  ```

### 数据库设计
1. **stores表**: 检索库信息 (id, scene, display_name, status等)
2. **objects表**: 检索对象信息 (source_path_original, managed_relpath等)
3. **vectors表**: 向量数据 (embedding_json存储向量副本)
4. **jobs表**: 异步任务信息 (status, progress等)

- 主启动脚本: `run.sh`

  - 网关: 18080
