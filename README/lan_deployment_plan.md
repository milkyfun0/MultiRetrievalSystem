# 多模态检索系统局域网部署整改方案

## 1. 产品需求文档(PRD)

### 1.1 项目背景
当前多模态检索系统采用本地部署模式，由于本地算力不足，需要将算法端和后端迁移到局域网环境。目的：因为在局域网系统演示中，因为演示电脑的性能，无法支撑起
后端和算法端的运行，所以先将算法端和后端迁移到局域网环境，演示机仅需要运行前端即可，前端仅需要调用局域网环境的后端的ip+端口。

### 1.2 核心目标
- 将计算密集型算法服务和后端迁移到局域网专门机器上
- 保持现有系统功能和接口不变
- 简化演示和测试流程

## 2. 技术实现方案
